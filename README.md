# supreme-octo-sniffle

The goal of Supreme Octo Sniffle is to create a high quality learning platform
that provides the few sought after features professors have been looking for.
The Supreme Octo Sniffle platform is developed under a set of general
guidelines.

- Should be free to use for both professors and students
- Should be open-source to foster code collaboration for the betterment of the
  platform
- Should provide enough tools to aid in different course teaching styles and
  course material
- Should be easily extensible to allow for new features to be implemented
  quickly
- Should minimize the number resources that depends on access to this platform
